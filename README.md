"# MissBsDreamCN" 

项目基本说明：

游戏类型：解谜脱出类小游戏
制作软件：RPGMaker MV
游戏时长：20~30分钟
结局数量：一个TE，三个NE
剧本、制作：寒冰封月
剧情介绍：那一天，我睡在阳台上我专属的座椅上，享受着正午的日光暖洋洋地洒在我的身上。温暖的感觉让我忍不住沉沉地睡去，直到……
公式网站：http://sirenisland.lofter.com
